## Testing the App

### Initial Setup
1. Launch the app
2. Tap "Sign in with Apple" on the welcome screen
3. Use your Apple Review Apple ID to authenticate
4. Grant permissions when prompted

### Key Features to Test

#### 1. Library & AI Search
- Navigate to "My Library" 
- Switch between "Movies & TV" and "Music" tabs
- Use the search bar to enter queries (e.g., "blink", "taylor swift", "action movies")
- Tap the "AI Search" button to see curated results
- Tap the heart icon to favorite/unfavorite items
- Verify favorites appear in the "Favorite Tracks" or "My Favorites" section

#### 2. Room Management
- Create a new room with a custom name
- View available rooms in the lobby
- Join an existing room
- Test room controls (if host)

#### 3. SharePlay Features
**Note:** SharePlay features require multiple devices on the same Apple ID or FaceTime call

- **Apple TV+**: Watch synchronized content together
- **Apple Music**: Listen to music in sync with others
- **Gaming**: Play Texas Hold'em with other users

#### 4. Multi-Platform
This app supports:
- iOS 17.0+
- macOS 14.0+
- tvOS 17.0+
- visionOS 1.0+

Test the app on different platforms to verify consistent experience.

## Privacy & Permissions

The app requests the following permissions:
- **Sign in with Apple**: Required for authentication
- **SharePlay**: Required for group experiences (optional, only when joining a SharePlay session)

## Known Limitations

- SharePlay features require an active FaceTime call or multiple devices signed in with the same Apple ID
- Some features (like actual video/audio playback) use placeholder data for demonstration purposes
- AI Search results are curated based on query keywords (not connected to a live AI API)
